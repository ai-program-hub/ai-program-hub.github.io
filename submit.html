📮 提交你的程序

分享你的 AI 创作，让更多人受益

如何提交程序？

1

准备你的程序

确保你的程序包含以下文件：

  • README.md - 程序说明（使用标准模板）
  • main.py (或其他) - 主代码文件
  • ai-metadata.json - AI 元数据（重要！）
  • requirements.txt - 依赖列表（如有）
2

创建 GitHub 仓库

  1. 访问 GitHub
  2. 仓库名：使用英文（如 my-awesome-tool）
  3. 选择组织：ai-program-hub（需要邀请）
  4. 勾选 "Add a README file"
  5. 点击 "Create repository"
3

上传代码

cd 你的程序文件夹
git init
git add .
git commit -m "Initial commit: 你的程序名称"
git remote add origin https://github.com/ai-program-hub/你的仓库名.git
git branch -M main
git push -u origin main
4

完成！

你的程序已经发布到 AI Program Hub！程序会自动出现在程序库中。

  "ai_readable": true
}

❓ 常见问题

Q: 什么样的程序可以提交？

A: 任何由 AI 生成的程序都可以！工具、游戏、脚本、自动化程序等。确保程序能正常运行，并有清晰的使用说明。

Q: 必须使用 MIT 许可证吗？

A: 推荐使用 MIT 许可证，但你可以选择任何开源许可证。请在 README 中明确说明。

Q: 我可以更新已提交的程序吗？

A: 当然可以！直接 push 更新到你的仓库即可，网站会自动同步。

Q: 如何获得组织成员资格？

A: 联系组织管理员，或在 GitHub 上提交一个 issue 说明你想贡献。
