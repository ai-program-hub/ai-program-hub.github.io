# 启用 GitHub Pages 的步骤

## 🌐 访问仓库设置

打开以下链接：
```
https://github.com/ai-program-hub/ai-program-hub.github.io/settings/pages
```

## 配置 Pages

1. **Source** 部分：
   - Branch: 选择 **master**
   - Folder: 选择 **/** (root)

2. 点击 **Save**

3. 等待 1-2 分钟

## ✅ 网站上线

网站会在以下地址上线：
```
https://ai-program-hub.github.io
```

## 📱 访问网站

上线后可以直接访问：
- 首页：https://ai-program-hub.github.io
- 程序库：https://ai-program-hub.github.io/programs.html
- 提交指南：https://ai-program-hub.github.io/submit.html

---

**提示**: GitHub Pages 首次部署可能需要 2-3 分钟。
刷新设置页面可以看到部署状态。
