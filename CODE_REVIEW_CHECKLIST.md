# 🔍 代码审查清单

提交程序前，请确保完成以下检查：

---

## ✅ 基础检查

- [ ] README.md 完整（使用标准模板）
- [ ] ai-metadata.json 填写完整
- [ ] main.py 能正常运行
- [ ] requirements.txt 列出所有依赖
- [ ] 有使用示例

---

## 🔐 安全检查

### 代码安全
- [ ] 无硬编码密钥/密码/Token
- [ ] 无未声明的网络请求
- [ ] 无系统命令执行（os.system, subprocess 等）
- [ ] 无访问系统敏感文件
- [ ] 文件操作有路径验证

### 依赖安全
- [ ] 依赖版本明确指定
- [ ] 无不必要的依赖
- [ ] 优先使用标准库
- [ ] 依赖包是官方 PyPI 包

### 数据安全
- [ ] 不收集用户数据（或明确说明）
- [ ] 不上传用户文件到外部
- [ ] 处理敏感数据有提示
- [ ] 输入验证完整

---

## 📝 文档检查

- [ ] 功能描述清晰
- [ ] 输入输出格式说明
- [ ] 使用示例完整
- [ ] 安全风险提示
- [ ] AI 生成信息完整（模型、日期）
- [ ] 标签分类准确

---

## 🧪 功能测试

- [ ] 程序能正常启动
- [ ] 示例能正常运行
- [ ] 错误处理合理
- [ ] 边界条件测试
- [ ] 无崩溃或死循环

---

## 🎯 质量标准

- [ ] 代码有基本注释
- [ ] 变量命名清晰
- [ ] 函数职责单一
- [ ] 代码长度合理（<500 行）
- [ ] 无重复代码

---

## 📋 提交前最后确认

- [ ] 已阅读 SECURITY.md
- [ ] 已阅读 CONTRIBUTING.md
- [ ] 同意 MIT 许可证
- [ ] 确认程序是原创或 AI 生成
- [ ] 无版权问题

---

## 🚀 审查流程

1. **自动检查** - GitHub Actions（待添加）
